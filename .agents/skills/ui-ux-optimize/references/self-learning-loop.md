# Self-Learning Loop Reference

The skill improves by logging insights and failures.

## Process
1. **Observe:** Identify a layout or design issue.
2. **Solve:** Implement a fix using tokens or structure.
3. **Log:** Record the lesson in `lessons_learned.md`.
4. **Apply:** Use the lesson in the next optimization run.

## Goal
Continuous improvement of design stability and fidelity.
