# Game UI Rules Reference

Game interfaces prioritize immersion and high-speed feedback.

## Principles
- **HUD (Heads-Up Display):** Peripheral placement for stats.
- **Haptics:** Visual feedback for every interaction (scale, glow).
- **Parallax:** Depth through Z-axis layering.
- **Safe Areas:** 24px offsets for HUD anchors to avoid thumb occlusion.
