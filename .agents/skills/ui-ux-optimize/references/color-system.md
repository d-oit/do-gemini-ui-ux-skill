# Color System Reference

A cohesive color palette defines the mood and brand of the application.

## Palette Structure
- **Background:** The base layer (e.g., deep charcoal, off-white).
- **Surface:** Elevated layers (cards, panels).
- **Primary:** Brand color for key actions.
- **Accent:** High-contrast color for highlights and status.
- **Muted:** Low-contrast colors for secondary information.

## Accessibility
- Ensure WCAG 2.1 AA compliance for text contrast.
- Use semantic colors for status (Success: Emerald, Error: Rose, Warning: Amber).
