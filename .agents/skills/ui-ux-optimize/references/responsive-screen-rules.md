# Responsive Rules Reference

Interfaces must adapt gracefully across all viewports.

## Breakpoints
- **Mobile:** < 768px (Single column, high density).
- **Tablet:** 768px - 1024px (Multi-column, relaxed spacing).
- **Desktop:** > 1024px (Full layout, immersive effects).

## Strategies
- **Fluid Typography:** Use `text-2xl md:text-4xl`.
- **Stacking:** Stack columns vertically on mobile.
- **Clearance:** Add extra padding/margins for sticky elements on mobile.
