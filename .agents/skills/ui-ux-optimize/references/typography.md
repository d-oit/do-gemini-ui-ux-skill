# Typography Reference

Typography is the foundation of information hierarchy.

## Font Selection
- **Display:** High-impact, unique fonts for headings.
- **Sans:** Clean, readable fonts for UI and body text.
- **Mono:** For technical data, code, and micro-labels.

## Scale
- **Heading:** `text-3xl` to `text-8xl`.
- **Body:** `text-sm` to `text-base`.
- **Micro:** `text-[10px]` with high tracking.

## Rules
- Avoid generic defaults (Inter, Roboto).
- Use `leading-tight` for large headings and `leading-relaxed` for body text.
- Enforce high contrast between heading and body fonts.
