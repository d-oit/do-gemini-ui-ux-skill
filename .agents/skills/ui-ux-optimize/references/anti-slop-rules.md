# Anti-Slop Rules Reference

"Slop" is generic, uninspired AI design. Avoid it.

## Anti-Slop Checklist
- [ ] No generic purple gradients on white.
- [ ] No default Inter/Roboto fonts.
- [ ] No cookie-cutter card layouts.
- [ ] No Timid palettes; use bold accents.
- [ ] No simulated infrastructure (fake loading states).

## Solution
Use opinionated design tokens and unique typography pairings.
