# Review Checklist Reference

A final check before completing a UI/UX task.

## Checklist
- [ ] Are all styles derived from tokens?
- [ ] Is the layout responsive (375px to 1440px)?
- [ ] Is there any flickering during transitions?
- [ ] Are there any overlapping elements?
- [ ] Is the typography readable and high-contrast?
- [ ] Does the design avoid "AI slop"?
