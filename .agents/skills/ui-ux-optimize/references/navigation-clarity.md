# Navigation Clarity Reference

Navigation should be intuitive and persistent.

## Models
- **Floating Header:** Glassmorphic bars at the top.
- **Sidebar:** For complex, multi-level IA.
- **Tab Bar:** For mobile-first primary navigation.

## Rules
- **Active State:** Clearly highlight the current location.
- **Mobile Nav:** Use `overflow-x-auto` for horizontal navigation bars on small screens.
- **Breadcrumbs:** For deep hierarchies.
