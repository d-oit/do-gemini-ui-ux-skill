# Layout & Composition Reference

Layout defines the structural relationship between elements.

## Grid Systems
- Use **Bento Grids** for multi-format content.
- Use **Flexbox** for linear alignments.
- Use **CSS Grid** for complex 2D layouts.

## Principles
- **Hierarchy:** Use size and weight to guide the eye.
- **Whitespace:** Use generous padding to prevent "sludge" aesthetics.
- **Rhythm:** Maintain consistent spacing tokens (`gap-4`, `gap-8`).
- **Anti-Overlap:** Avoid absolute positioning for core content.
