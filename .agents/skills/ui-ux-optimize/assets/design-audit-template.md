# Design Audit Template

Use this template to identify layout risks and design slop.

## Audit Checklist
### 1. Stability
- [ ] Anti-flicker tokens applied?
- [ ] Overflow-x-hidden on root?

### 2. Responsiveness
- [ ] Mobile nav scrollable?
- [ ] No text cutoffs at 375px?

### 3. Aesthetics
- [ ] Unique fonts used?
- [ ] High-contrast palette?
- [ ] No generic AI gradients?
